# Student Performance. Feedback System

## Overview

This project is a Python application that analyzes student performance. It calculates CGPA compares marks with class averages provides feedback for each subject and visualizes performance using graphs.

The system helps students understand their strengths and weaknesses and identify areas for improvement.

## Problem Statement

Students often do not understand how their academic performance compares with their peers. This makes it hard for them to identify subjects and improve.

## Solution

This project solves the problem by:

* Calculating CGPA based on subject marks

* Comparing performance with class averages

* Providing subject- feedback

* Predicting performance level

* Identifying subjects

* Visualizing performance using graphs

## Methodology

1. User inputs subject names, marks and class averages

2. System calculates grade points. Cgpa

3. Statistical analysis is performed

4. Rule-based logic generates feedback and predictions

5. Data is visualized using a bar graph
